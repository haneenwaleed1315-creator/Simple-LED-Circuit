
# Simple LED Circuit

## Overview
This project demonstrates a simple LED circuit built on a breadboard using basic electronic components.

## Components
- Breadboard
- 9V Battery
- LED
- 220Ω Resistor
- Jumper Wires


## What I Learned
- Building a basic electronic circuit
- Understanding current flow
- Using a resistor to protect an LED
- Breadboard connections
